Block Diagram Of Our System


  • • User Input: The user enters the name of a movie they like or preferred.
  • • Data Preprocessing: The system retrieves information about the movies from a datasets, like cast, geners, crew and keywords . The data is cleaned and transformed to ensure that it is in a format suitable for features extraction
  • • Features Extraction: The system extracts features from the movie data, such as keyword, cast, crew and genres, that will be calculated similarity scores between movies
  • • Cosine Similarity: The system calculates the cosine similarity between the features of the input movie and the features of all the other movies in the dataset. This produces a similarity scores for each movie in the datasets
  • • Top-N Recommendation: The system selects the top-N movies with the highest similarity scores and recommends them to the users.

Algorithm Details

Count Vectorizer
  • Using count vectorization, our system "MOVFLIX" can compare the count vectors of different movies to determine which movies are most similar in terms of their content, and make recommendations accordingly.
    In other words, if the count vector of a movie d is represented as CV(d), then
    CV(d)[i] = frequency of word i in the summary or review of movie d
    For instance,
    If the movie "The Dark Knight" is described as "Batman must battle the Joker to save Gotham City," and
    V = "action," "thriller," "drama," "hero," and "villain," then the count vector for "The Dark Knight" would be:
    "The Dark Knight" CV = 0 1 1 1 This indicates that "The Dark Knight" has no instances of "action" but does have one instance of "thriller," one instance of "drama," and one instance of each of "hero" and "villain."
Cosine Similarity
  • The higher the cosine similarity value, the more similar the movies are consider to be, and therefore the more likely that a user who like one movie will also like the other.     The dot product of the two features vectors (V1.V2) measures the similarity between the movies in terms of their shared features.
    while the denominator (||V1||.||V2||) normalizes the results to account for difference in the magnitudes of the features vectors.
    The resulting cosine similarity value ranges from -1 to 1, where a value of 1 indicates that the movies are identical, 0 indicates that they are completely dissimilar, and -1 indicate that they are completely opposite.
